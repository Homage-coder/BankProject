# BankProject

A Solidity smart contract project that implements a simple banking system and demonstrates the use of arrays and for loops.

## Features
- Account creation
- Deposits
- Withdrawals
- Balance tracking
- Multiple account registration using for loops

## Contracts
- `BankProject1.sol`
- `ForLoop.sol`

## Technologies Used
- Solidity ^0.8.24
- Remix IDE / Foundry
